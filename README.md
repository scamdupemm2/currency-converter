## Описание программы

Currency Converter — это графическое приложение для конвертации валют с использованием внешнего API. Позволяет выполнять конвертацию между различными валютами, сохранять историю операций в JSON-файл и просматривать историю в удобной таблице.

---

## Требования для запуска

Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `pip install requests`

---

## Как запустить

```bash
git clone https://github.com/ваш-логин/currency-converter.git
cd currency-converter
python app.py
```

---

## Как пользоваться

1. При запуске появляется интерфейс с полями ввода
2. Введите сумму в поле "Сумма"
3. Выберите валюту "Из" и "В" из выпадающих списков
4. Нажмите кнопку "Конвертировать"
5. Результат отобразится на экране
6. История конвертаций сохраняется автоматически и отображается в таблице ниже

---

## Пример работы

Конвертация 100 USD в RUB:
- Сумма: 100
- Из: USD
- В: RUB
- Результат: 100 USD = 7499.59 RUB

История сохраняется в файл `data/history.json`.

---

## Тестирование

| Тип теста | Пример | Результат |
|-----------|--------|-----------|
| Позитивный | Ввод 100 USD -> RUB | ✅ Конвертация выполнена |
| Негативный | Ввод "abc" вместо суммы | ❌ Ошибка: введите корректное число |
| Граничный | Ввод 0 | ❌ Ошибка: сумма должна быть больше 0 |
